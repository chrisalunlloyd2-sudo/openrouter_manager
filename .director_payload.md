## Sidebar Navigation
The sidebar navigation will be implemented using a recursive template to ensure consistency across all pages.

### Template Structure
The template will consist of the following elements:
- **Header**: Contains the project logo and title
- **Navigation Menu**: A list of links to all pages in the project
- **Footer**: Contains copyright information and links to external resources

### Navigation Menu
The navigation menu will be generated dynamically using a recursive function to traverse the project's directory structure.

[FILE: openrouter_manager/templates/sidebar_navigation.html]
```html
<div class="sidebar-navigation">
  <div class="header">
    <img src="logo.png" alt="Project Logo">
    <h1>Open Router Manager</h1>
  </div>
  <ul class="navigation-menu">
    {% for page in pages %}
      <li><a href="{{ page.url }}">{{ page.title }}</a></li>
    {% endfor %}
  </ul>
  <div class="footer">
    <p>&copy; 2023 Open Router Manager</p>
    <ul>
      <li><a href="https://github.com/chrisalunlloyd2-sudo/openrouter_manager">GitHub Repository</a></li>
    </ul>
  </div>
</div>
```

[FILE: openrouter_manager/utils/navigation.py]
```python
import os

def get_pages():
    pages = []
    for root, dirs, files in os.walk("."):
        for file in files:
            if file.endswith(".md"):
                page = {
                    "title": file.replace(".md", ""),
                    "url": file.replace(".md", ".html")
                }
                pages.append(page)
    return pages
```
